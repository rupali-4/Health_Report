Welcome To LogicLoom

A hierarchical target planning and tracking system is to be built to visualize and report on how project targets assigned by the PMO flow down through VPs → RMs → Centers → Batches, and how actual enrollments match those targets. This tool is to support multiple programs like Deep Tech, SAVE, BEST, Diya, etc. and must be capable of generating multi-dimensional reports.

Features

MULTIUSER

Allows multiple people to access and use
the system, each with their own account
or role.

Project Creation and Management

Enables users to start new projects,
set details, assign tasks, and
monitor progress over time.

Report Generation

Creates organized summaries of data
in formats like charts or tables for
analysis.

Project Target Allocation

Distributes specific goals or objectives
to teams or individuals within a project
for better tracking and accountability.
